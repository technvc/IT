เรียนจบแผนกนี้ทำอาชีพอะไรได้บ้าง

โปรแกรมเมอร์ (Programmer)

PROGRAMMER คือผู้นำข้อมูลไปออกแบบรายละเอียดการวางโครงสร้างระบบคอมพิวเตอร์ โดยเขียนโปรแกรมด้วยภาษาทางคอมพิวเตอร์ที่แตกต่างกัน มีหน้าที่เขียนและทดสอบรหัสหรือโค้ดเพื่อทำให้คอมพิวเตอร์และซอฟต์แวร์ทำงานได้ และทำการตรวจสอบรหัสที่เกิดข้อผิดพลาดหรือซ่อมแซมแก้ไข ในกรณีที่เกิดปัญหาจากการทำงาน

นักพัฒนาเว็บไซต์ (Web Developer)

Web Developers คือ ผู้พัฒนาเว็บไซต์ เป็นตำแหน่งงานที่มีหน้าที่ครอบคลุมเกี่ยวกับการดูแลเว็บไซต์ทั้งหมด ยกเว้นเรื่องการออกแบบเว็บไซต์

Web Designer

คือผู้ที่ทำงานด้านออกแบบเว็บไซต์ ให้ออกมาสวยงาม ใช้งานได้ง่าย ตรงกับความต้องการของลูกค้า และตอบสนองต่อการใช้งานได้ดี

Web Master

คือผู้ที่ดูแลภาพลักษณ์โดยรวมของเว็บไซต์ กำหนดทิศทางของเว็บไซต์ ดูแลความเคลื่อนไหว เพิ่ม ลด ฟังก์ชั่นและฟีเจอร์ต่าง ๆ ของเว็บไซต์

ผู้บริหารจัดการระบบฐานข้อมูล

มีหน้าที่เป็นสื่อกลางระหว่างผู้ใช้กับโปรแกรมเมอร์ ในการจัดการ Database Management ให้ผู้ใช้เข้าถึงข้อมูลได้สะดวกง่ายดาย มีประสิทธิภาพ

Systems Analysis หรือนักวิเคราห์ระบบงาน

จะเป็นผู้วิเคราะห์โครงการธุรกิจต่าง ๆ ที่จะนำการประมวลผลด้วยคอมพิวเตอร์เข้าช่วยในการกำหนดผังการทำงาน ตระเตรียมข้อมูลและขั้นตอน เพื่อช่วยให้ได้ผลการดำเนินการธุรกิจตามต้องการ

IT / Network Support หรือผู้สนับสนุนและตรวจสอบระบบสารสนเทศ

คือผู้ที่มีหน้าที่ดูแลรับผิดชอบ แก้ไขจัดการ บำรุงรักษา และซ่อมแซมระบบ Software และ Hardware ขององค์กร ตลอดจนอุปกรณ์คอมพิวเตอร์ขององค์กรให้พร้อมใช้งานและอยู่ในสภาพดีตลอดเวลา

Systems Engineer หรือ วิศวกรสารสนเทศ

มีหน้าที่รับผิดชอบเกี่ยวกับการทำงานของระบบคอมพิวเตอร์ ออกแบบ ชี้แนะหรือวางข้อกำหนดในการซื้อหาเครื่องใช้ให้เหมาะสมกับระบบ

Technical Consultant หรือ ผู้จัดการ / ที่ปรึกษาด้านเทคโนโลยีสารสนเทศ

มีหน้าที่ดูแลรักษาระบบ แก้ปัญหาทางเทคนิคของเครือข่ายทั้ง Server และระบบ Internet ขององค์กร

SEO (Search Engine Optimization) Analysis

คือผู้ที่ทำงานเกี่ยวข้องกับการทำให้เว็บไซต์ขององค์กรติดอันดับอยู่ในลำดับต้นๆ ใน Search Engine ต่างๆ เช่น Google , Yahoo , Bing เป็นต้น จำเป็นต้องมีความรู้เกี่ยวกับการสร้าง คิดแผนการตลาดทางเว็บไซต์ ออกแบบกราฟฟิกต่าง ๆ และค้นหาด้วยคีย์เวิร์ด

นักเขียนโปรแกรมบนมือถือ

คือผู้ออกแบบและพัฒนา Application ต่างๆ บนโทรศัพท์มือถือ

นักพัฒนาเกมส์

คือผู้ออกแบบ สร้าง และพัฒนาเกมส์ออนไลน์
